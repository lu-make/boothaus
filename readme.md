# Boothaus

Eine einfache Lagerverwaltungssoftware speziell für Winterlager für Sport- und Freizeitboote

## Funktionsumfang

Die Software erlaubt die Verwaltung von Booten sowie die Verteilung von Lageraufträgen und Lagerplätzen pro Wintersaison. Es gibt eine zweidimensionale Übersicht über Lagerplätze
als Raster und die Möglichkeit, Lageraufträge automatisch auf freie Lagerplätze zu verteilen.
